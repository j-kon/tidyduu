# TidyDuu Portfolio Screenshots 📸

Place your high-quality, trimmed application screenshots in this folder to display them inside the main repository `README.md`.

## Recommended Screenshots List

To show off the features of TidyDuu on your GitHub and portfolio, capture the following states:

1. **`home_light.png`**: The main Tasks tab in light mode (showing active and completed task items).
2. **`home_dark.png`**: The main Tasks tab in dark mode showing the modern Material 3 dark palette.
3. **`today_view.png`**: Today Focus tab showcasing the progress card and focus-specific list.
4. **`calendar_view.png`**: Calendar tab displaying the month day-grid with task indicators.
5. **`add_task_dialog.png`**: The "New Task" bottom sheet form showing dropdown selections and option switches.

## Capturing Guidelines

- **Device Frame**: For professional results, wrap screenshots in clean device frames (e.g., using services like [Deviceframes](https://deviceframes.com/) or [Previewed](https://previewed.app/)).
- **Resolution**: Keep resolution clean (recommended height of 800px to 1200px) and compress images to `.png` or `.webp` format to keep loading speeds lightning-fast on mobile and desktop web.
- **Dimensions**: Ensure consistent aspect ratios across screenshots.
