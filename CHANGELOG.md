# Changelog

All notable changes to TidyDuu will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-06-25

### Added
- **Core Task Management**: Create, edit, search, categorise, toggle, and delete todos.
- **Swipe Actions**: Swipe right to toggle completion status, swipe left to prompt deletion with Undo restoration.
- **Subtasks Support**: Optional checklists for tasks with completion progress metrics.
- **Recurring Tasks**: Options for daily, weekly, and monthly recurrence shifting task dates forward automatically.
- **Task Notes**: Support persistent reference notes displayed as card indicators.
- **Dashboard Screen**: Aggregated completion stats, overdue and high-priority counters, completion rate rings, and daily productivity streaks.
- **Focus Mode**: Pomodoro timer (25 min focus / 5 min break) featuring circular progress rings, local alarm notifications, and subtask checklists.
- **Quick Add Bar**: Smooth expanding text field for fast task creation directly from the home list.
- **Settings Screen**: Management of light/dark/system theme selection, default task priorities/reminders, notification alerts configurations, and task cleanup tools.
- **UI/UX Polish**: Beautiful purple branding design system utilizing staggered card animations and tab transitions.
